¡Bienvenidos a la sección dos! Si bien la sección uno se centró por completo en los fundamentos iniciales de PHP, ahora continuaremos y aprenderemos cómo se ve construir una aplicación PHP típica con una base de datos MySQL.

## Cosas que aprenderemos
- Page Linking (Ligas a paginas)
- Templates (Plantillas)
- TailwindCSS

## Escoger una plantilla de Tailwind
Por ejemplo [esta](https://tailwindui.com/components/application-ui/application-shells/stacked) y copiar el código dentro de `<body>` de nuestra vista `index.view.php` Ver las instrucciones que trae la plantilla de tailwind.
```php
<html class="h-full bg-gray-100">
<body class="h-full"> 
```
Y aplicarla, también incluir las librerías `cdn` de de `tailwindcss` con:
```php
<script src="https://cdn.tailwindcss.com"></script> 
```
En este capitulo nos vamos a ir por el camino largo, copiando directamente todo el html de la vista principal a todas las demás vistas que deseamos crear. 
## Vamos a crear las siguientes vistas
- index.view.php la mandamos llamar con index.php
- about.view.php la mandamos llamar con about.php
- contact.view.php la mandamos llamar con contact.php

## En index.php
```php
require "index.view.php"; 
```
## En about.php
```php
require "about.view.php"; 
```
## En contact.php
```php
require "contact.view.php"; 
```
